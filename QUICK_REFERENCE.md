**Print this page for your desk!**

## 🎯 Common Tasks - At a Glance

### Creating a Post (2 minutes)

1. Click **➕ Create Post**
2. (Optional) Select template from dropdown
3. Fill required fields (marked with *):
   - Client
   - Post Copy
   - Scheduled Date
4. Select platforms → Add Box.com image links
5. ✅ Check "Share with client" if client should see this
6. Click **Submit for Internal Review** (or save as draft)

**Remember**: Upload images to Box.com FIRST, then paste link!

---

### Approving a Post (30 seconds)

1. Click **📥 My Approvals** badge
2. Click post to open details
3. Review copy and images
4. Add comment (optional)
5. Click **✅ Approve** or **🔄 Request Changes**

**Tip**: Request changes with specific feedback, not just "fix this"

---

### Editing a Post (3 minutes)

1. Click post on calendar
2. Click **Edit Post** button
3. Make changes
4. ✅ Check "Share with client" if they should see this edit
5. Click **Save** or **Submit for Review**

**Note**: Editing in Client Review returns post to Draft

---

### Creating a Template (1 minute)

1. Open any post detail view
2. Click **📋 Save as Template**
3. Enter template name (e.g., "Holiday Post - LinkedIn")
4. Click OK

**Use templates** for recurring content like holidays, weekly tips, product launches

---

### Using a Template (1 minute)

**Method 1**: From post creation form
- Select template from "Start from Template" dropdown

**Method 2**: From template library
- Click **📋 Templates** → Click **✏️ Use** on template

Then fill in date, client, and images as normal.

## 📊 Status Quick Reference

| Color | Status | What It Means | What to Do |
|-------|--------|---------------|------------|
| ⚪ Gray | Draft | Being created | Keep editing or submit for review |
| 🟠 Orange | Internal Review | Team reviewing | Wait for feedback or approve if you're an approver |
| 🔴 Red | Client Review | Client reviewing | Wait for client or follow up if delayed |
| 🟢 Green | Approved | Ready to publish | Schedule or publish |
| 🟦 Blue | Scheduled | Queued to post | Will post automatically on date |
| 🟪 Purple | Published | Already posted | Review performance |

## ⚠️ Common Mistakes to Avoid

**❌ DON'T:**
- Submit without uploading images to Box first
- Forget to select platforms
- Skip internal review for important posts
- Edit posts without checking "Share with client" box
- Approve without reviewing images
- Delete posts (change status to Draft instead)

**✅ DO:**
- Double-check scheduled dates
- Use templates for recurring content
- Add notes for context
- Review version history before editing
- Check Box links work before submitting
- Use specific feedback when requesting changes

## 🚨 Emergency Procedures

### "I accidentally deleted a post!"

1. Don't panic - it's recoverable
2. Contact admin immediately
3. Provide post title or approximate date
4. Admin can restore from Google Sheets version history

### "Client needs urgent changes to already-approved post!"

1. Open post → Click **Edit Post**
2. Make changes
3. **Uncheck** "Share with client" (if internal fix)
4. Save as Draft
5. Resubmit for client review if needed

### "Images not loading!"

1. Verify Box.com link is a shared link (not private)
2. Test Box link in new browser tab
3. Check link starts with `https://`
4. Re-copy link from Box.com
5. Try refreshing page (Ctrl+R / Cmd+R)

### "Can't submit for Client Review - validation error!"

**Error**: "Internal approvals must be completed first"

**Solution**:
- Wait for all internal approvers to approve
- OR admin can bypass and submit directly

## 📸 Box.com Quick Guide

### Getting a Shared Link

1. Upload image/video to Box.com
2. Right-click file → **Share**
3. Click **Create shared link**
4. Copy link
5. Paste in Social Media Planner

### For Carousels (Multiple Images)

1. Create folder in Box.com
2. Upload all carousel images to folder
3. Right-click folder → **Share**
4. Click **Create shared link**
5. Copy folder link
6. Paste in Social Media Planner
7. Post shows 📸 carousel indicator

## 🗂️ File Naming Conventions

**For Box.com uploads:**
- Use descriptive names: `HEMIC_Holiday_LinkedIn_Image1.jpg`
- Include client, campaign, platform
- Avoid special characters except dash and underscore
- Use consistent date format if including dates: `YYYYMMDD`

**Good Examples:**
- ✅ `EPIC_Product_Launch_Facebook_Hero.png`
- ✅ `Salvation_Army_Holiday_2025_Carousel_1.jpg`
- ✅ `HIMI_Weekly_Tip_LinkedIn.mp4`

**Bad Examples:**
- ❌ `Image1.jpg` (too vague)
- ❌ `final version 2 UPDATED.png` (spaces, unclear)
- ❌ `IMG_2024.jpg` (no context)
